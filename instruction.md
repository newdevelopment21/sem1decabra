# **Инструкция по использованию системы контроля версий  Git**

## Что такое git.

git Это ..

## Инициализация репозитория

чтобы инициализировать, то есть создать репозиторий, нужно ввести в терминале команду гит инит
    
    git init

## Проверка состояния репозитория

    чтобы проверить состояние репозитория, нужно ввести 

        git status

## Добавление изменений в индекс

    чтобы внести изменеия для следующего коммита нужно сначала ввести в терминале команду гит эдд:

        git add <failname>
